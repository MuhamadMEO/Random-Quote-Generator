# Random-Quote-Generator

**Автор:** Мухамад Курманов

## Описание
Приложение для генерации случайных цитат с возможностью фильтрации по автору и теме, а также сохранением истории в файл.

## Использование
1. Запустите `gui.py`.
2. Нажмите «Сгенерировать цитату» для получения случайной цитаты.
3. Используйте поля для фильтрации истории по автору или теме.
4. История сохраняется в `history.json`.

## Примеры тестов
- Генерация нескольких цитат и проверка их отображения в истории.
- Фильтрация по автору «Джон Леннон».
- Проверка сохранения и загрузки истории после перезапуска.

## Запуск приложения
1. Убедитесь, что Python установлен.
2. Перейдите в папку проекта через терминал или командную строку.
3. Запустите приложение командой:
   ```
   python gui.py
   ```
