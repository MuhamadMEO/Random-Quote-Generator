# Random-Quote-Generator

**Автор:** Мухамад Курманов

## Описание

Приложение для генерации случайных цитат с возможностью просмотра истории и фильтрации по автору и теме.

### Функциональность:
* Генерация случайных цитат из предопределённого списка
* Сохранение истории сгенерированных цитат
* Фильтрация истории по автору и теме
* Загрузка цитат и истории из JSON‑файлов
* Визуальный интерфейс на Tkinter

### Технические детали:
* Язык: Python 3
* GUI‑библиотека: Tkinter
* Хранение данных: JSON‑файлы
* Зависимости: стандартные библиотеки Python
